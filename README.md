# Smart Study Planner (Java)

## Problem Statement
Students often struggle to manage study tasks efficiently and track academic progress. This project provides a console-based Smart Study Planner that helps organize tasks, monitor completion status, detect weak subjects, and suggest what to study next.

## Features
- Add study tasks
- View pending tasks
- View completed tasks
- Mark tasks as completed
- Detect weak subjects automatically
- Study recommendation system
- Performance analytics (completion percentage)
- File-based data storage using planner.txt

## Technologies Used
- Java
- Object-Oriented Programming
- ArrayList
- HashMap
- File Handling
- Menu-driven interface

## How to Run

Compile the program:

javac src/*.java

Run the program:

java -cp src Main
